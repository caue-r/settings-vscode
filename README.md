# VSCode Visual Settings

Este repositório contém as minhas configurações visuais personalizadas para o Visual Studio Code (VSCode), com ajustes de tema, fontes e layout.

## Passo a Passo

1. **Copie o conteúdo do `settings.json`:**
   - Abra o arquivo `settings.json` deste repositório.
   - Copie todo o conteúdo do arquivo.

2. **Cole no VSCode:**
   - No VSCode, pressione `Ctrl + Shift + P` para abrir a paleta de comandos.
   - Pesquise por **"User Settings (JSON)"** e selecione a opção.
   - Cole o conteúdo copiado no arquivo de configurações do VSCode.

3. **Instale as fontes:**
   - **JetBrains Mono**: [Baixe aqui](https://www.jetbrains.com/lp/mono/)
   - **Inter**: [Baixe aqui](https://fonts.google.com/specimen/Inter)

4. **Tema:**
   - Instale o tema **Min Theme** no VSCode:  
     [Min Theme - Visual Studio Marketplace](https://marketplace.visualstudio.com/items/?itemName=miguelsolorio.min-theme)

## Créditos

- **Fontes:**
  - **JetBrains Mono**: Desenvolvida pela JetBrains, uma fonte monoespaçada projetada para programadores. [Saiba mais](https://www.jetbrains.com/lp/mono/)
  - **Inter**: Fonte sans-serif projetada para interfaces de usuário e legibilidade em telas. Criada por Rasmus Andersson. [Saiba mais](https://fonts.google.com/specimen/Inter)
